# 食谱
## 1. 金枪鱼蔬菜全麦三明治
**食材准备**
- [ ] 全麦吐司
- [x] 水浸金枪鱼（鱼家乡）25.9/3罐
- [x] 煎蛋
- [ ] 黄瓜
- [ ] 胡萝卜
- [x] 蜂蜜芥末酱 盛源来 14.5/罐

## 2. 红薯泥火腿全麦三明治
食材：红薯泥+低脂火腿+煎蛋+蕃茄酱

## 3. 鸡蛋火腿蔬菜三明治
食材：火腿+鸡蛋+黄瓜+番茄

## 4. 牛油果鸡蛋全麦三明治
食材：煎蛋+番茄+牛油果+蜂蜜芥末酱

## 5. 番茄鸡蛋蔬菜三明治
食材：鸡蛋+番茄+生菜+蕃茄酱

## 6. 鸡胸紫薯三明治
食材：鸡蛋+鸡胸肉+生菜+紫薯+蜂蜜芥末酱

## 7. 肉松鸡肉蔬菜三明治
食材：鸡蛋+肉松+生菜+番茄+鸡胸肉+蜂蜜芥末酱
